# IDS-TPFINAL

el projecto consisite en construir una página web de hospedaje con el objetivo de facilitar la reserva de alojamientos y ofrecer al usuario información clara de disponibilidad, precios y servicios,  se debe mostrar claramente :
 - Página principal con breve descripción de la página 
 - Fotografías con detalles del hotel/ cabaña/ departamento 
 - Página de reserva 
 - Contactos ( comentarios y opiniones )
 - Cuenta del usuario (con Historial de visitas )
 - Mapa interactivo
